# Movie Recommendation System 🎬

## Problem
People often struggle to find movies based on their interests.

## Solution
This project recommends movies based on similarity using content-based filtering.

## How It Works
- Uses movie genres
- Converts text into vectors
- Calculates similarity using cosine similarity

## How to Run
1. Install Python
2. Install libraries:
   pip install pandas scikit-learn
3. Run:
   python main.py

## Example
Input: Avengers  
Output: Iron Man, Thor, Hulk...
