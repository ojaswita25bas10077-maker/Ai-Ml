# 🎬 Movie Recommendation System

## 📌 Problem
People often struggle to decide what movie to watch.

## 💡 Solution
This system recommends similar movies based on content using machine learning.

## ⚙️ How It Works
- Uses movie genres
- Converts text into vectors using CountVectorizer
- Calculates similarity using cosine similarity

## 🚀 How to Run
1. Install Python
2. Install libraries:
   pip install pandas scikit-learn
3. Run:
   python main.py

## 🎯 Example
Input: Avengers  
Output: Iron Man, Thor, Hulk...

## 🧠 Concepts Used
- Content-Based Filtering
- NLP (Text Vectorization)
- Cosine Similarity
Output: Iron Man, Thor, Hulk...
